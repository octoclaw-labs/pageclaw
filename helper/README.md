# PageClaw Native Helper

The Native Helper is planned for the installable beta. It will start and supervise the local Bridge so normal users do not need to run `node bridge/server.js` manually.

Current developer command:

```bash
PAGECLAW_TOKEN=pc_local_dev node bridge/server.js
```

Future scope:

- Native Messaging host manifest installer
- automatic Bridge start / stop
- tray status indicator
- version compatibility check
