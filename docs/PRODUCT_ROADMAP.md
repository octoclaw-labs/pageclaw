# PageClaw Product Roadmap

## North Star

Turn an already-signed-in web model page into a local OpenAI-compatible provider for PageAgent and other local AI clients.

## v0.19 — installable beta

- Native Helper installer for macOS / Windows / Linux
- one-click local Bridge start/stop
- first-run onboarding
- error recovery UI

## v0.20 — driver pack

- dedicated DeepSeek driver
- dedicated Qwen / Tongyi driver
- dedicated GLM / Zhipu driver
- dedicated Doubao driver
- fixtures for request replay and response parsing

## v0.21 — PageAgent native experience

- floating Copilot button for business pages
- one-click PageAgent injection
- visible action log
- confirmation before risky page operations

## v0.22 — Chrome Web Store RC

- permission minimization
- privacy policy
- store screenshots
- audit notes for local-only processing

## v1.0 — GA

- stable Provider API
- stable PageAgent integration
- stable export path
- native helper package
- complete user guide
